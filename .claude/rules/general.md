# General

- Never reference or link to internal or local-only documents (e.g., scratch
  notes under `~/nickolashkraus/agent-os/`, files on disk that are not in
  a public repo, private working docs) from external content (GitHub PRs,
  Notion pages, Linear issues, Slack messages, blog posts). Readers cannot
  resolve the link, and the reference leaks internal context. When the
  underlying detail is useful, inline a summary instead. Internal context
  belongs in memory or daily task files, not in external posts.
